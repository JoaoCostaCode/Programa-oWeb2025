# Plataforma Web com Jogo Integrado e Ranking  

Este projeto é uma plataforma web desenvolvida em Node.js, que integra um jogo interativo de guerra espacial diretamente no site.  
A aplicação permite que usuários joguem, salvem suas pontuações e disputem posições em um ranking online.  

## Funcionalidades  

-  Integração de jogo dentro do site  
-  Sistema de autenticação e gerenciamento de usuários  
-  Ranking atualizado em tempo real  
-  Salvamento de pontuações em banco de dados  
-  Interface web responsiva  

## Tecnologias Utilizadas  

- Backend: Node.js (Express)  
- Frontend: HTML, CSS, JavaScript  
- Banco de Dados: MySQL 
- Controle de Versão: Git & GitHub
